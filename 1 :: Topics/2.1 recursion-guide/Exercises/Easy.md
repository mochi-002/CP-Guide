# Easy Recursion Problems

1. **Reverse String**
   - [LeetCode 344](https://leetcode.com/problems/reverse-string/)
   - Key Concept: Base case + swapping first/last elements
   - Hint: `reverse(s) = last_char + reverse(remaining_string)`

2. **Fibonacci Number**
   - [LeetCode 509](https://leetcode.com/problems/fibonacci-number/)
   - Key Concept: Classic recursion introduction
   - Variants: Try both naive and memoized versions

3. **Power of Three**
   - [LeetCode 326](https://leetcode.com/problems/power-of-three/)
   - Key Concept: Recursive division
   - Hint: `n == 1` (base case), `n % 3 == 0` (recursive case)

4. **Binary Tree Inorder Traversal**
   - [LeetCode 94](https://leetcode.com/problems/binary-tree-inorder-traversal/)
   - Key Concept: Tree recursion pattern (Left → Root → Right)
