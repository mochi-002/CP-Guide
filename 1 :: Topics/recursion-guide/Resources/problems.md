### Tips for Solving ✨
1. **Starting Point** 🎯
   - Identify the base case
   - Draw recursive tree
   - Test with small inputs

2. **Debugging** 🔍
   - Print intermediate states
   - Track function calls
   - Use VS Code debugger

3. **Testing** 🖥️
```bash
g++ -o solution solution.cpp
./solution < input.txt > output.txt
```
